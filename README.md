# Консольное приложение для управления личной коллекцией фильмов.

Позволяет добавлять, просматривать, отмечать просмотренные фильмы и искать их по году выпуска.
Приложение можно запустить с цифры
Чтобы запустить программу:

```bash
  pip install -r app.py
```

```
Приложение можно запустить с помощью цифр из меню

1. Показать все фильмы
2. Показать непросмотренные фильмы
3. Добавить фильм
4. Отметить фильм как просмотренный (с оценкой или без)
5. Найти фильмы по году
0. Выход
```
